GEMC is a Monte Carlo tool based on `geant4 <https://geant4.cern.ch>`_  to simulate the passage of particles through matter, the hardware electronics of
detectors and their the digitized response.


 * application independent geometry description
 * easy interface to build / run experiments
 * cad/gdml imports
 * digitization and output plugins
