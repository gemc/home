# `show_field_lines`

Type: `option`

Description: number of field-line points to display

Generated from:

```sh
gemc help show_field_lines
```

```text
-show_field_lines=<value> ..: number of field-line points to display


   Display magnetic field lines in the initial visualization scene.
   
   The value is the number of field-line integration points. Set to 0 to disable.
   
   Example: -show_field_lines=100
```
