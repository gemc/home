# `fieldAt`

Type: `option`

Description: query all configured fields at x y z

Generated from:

```sh
gemc help fieldAt
```

```text
-fieldAt=<value> ...........: query all configured fields at x y z


   Evaluate all configured electromagnetic fields at one absolute coordinate.
   
   The value must contain three coordinate expressions with units, separated by spaces.
   
   Example: -fieldAt="10*cm 0*mm 2*m"
```
