# `tt`

Type: `option`

Description: GUI test timeout (ms)

Generated from:

```sh

```

```text
-tt=<value> ................: GUI test timeout (ms)


   Milliseconds a GUI-based test waits before it auto-closes, so the module
   example/test programs can run unattended in CI.
   
   Example: -tt=1000
```
