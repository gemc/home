# `useBackupMaterial`

Type: `option`

Description: Backup material

Generated from:


```text
-useBackupMaterial=<value> .: Backup material


   Material to be used if some volume requested a non-defined material.
   By default GEMC will exit with error.
   Example: -useBackupMaterial=G4_Air
```
