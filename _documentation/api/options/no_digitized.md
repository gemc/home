# `no_digitized`

Type: `option`

Description: detectors for which digitization is disabled

Generated from:

```sh
gemc help no_digitized
```

```text
-no_digitized=<value> ......: detectors for which digitization is disabled


   Disable digitization for a comma- or whitespace-separated list of detector names.
   Use "all" to disable digitization for every detector. Default: none.
   
   Example: -no_digitized="ftof, ecal"
```
