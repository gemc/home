# `no_true_info`

Type: `option`

Description: detectors for which true-information output is disabled

Generated from:

```sh
gemc help no_true_info
```

```text
-no_true_info=<value> ......: detectors for which true-information output is disabled


   Disable true-information output for a comma- or whitespace-separated list of detector names.
   Use "all" to disable true-information output for every detector. Default: none.
   
   Example: -no_true_info="ftof, ecal"
```
